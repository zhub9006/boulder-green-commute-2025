# Boulder Green Commute Guide 🌿

A sustainable travel guide for Boulder residents — walking and biking routes to help you commute smarter, healthier, and greener.

## Sample Commute

**From:** 45 Oak Avenue, Boulder, CO 80304 → **To:** 1800 Broadway, Boulder, CO 80302

| Mode | Distance | Time | Calories | CO2 Saved |
|---|---|---|---|---|
| Walking | 4.03 km (~2.5 mi) | ~48-50 min | 320-400 kcal | ~0.8 kg/day |
| Biking | 4.03 km (~2.5 mi) | ~16-20 min | 150-200 kcal | ~0.8 kg/day |

## Route Overview

Both routes follow the same path through Boulder:
1. Oak Avenue → south from home
2. 19th Street → main artery (2.16 km)
3. 20th Street → connects to downtown
4. Pearl Street → pedestrian mall area
5. 18th Street → short connect to Canyon
6. Canyon Boulevard → scenic path along Boulder Creek to office
